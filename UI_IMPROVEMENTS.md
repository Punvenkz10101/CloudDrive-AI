## Sections Explained

There are two main sections:
1. **AI Assistant (Chat Interface)** - Ask questions about your files
2. **File Management** - Upload, view, and manage files

Both serve different purposes:
- **Chat** = Interactive Q&A about file contents
- **Files** = File operations (upload, delete, download)

## Color Scheme

- **Primary:** `emerald-brand` (#10b981 or similar)
- **Text:** `charcoal` (dark gray)
- **Background:** `beige` (light background)
- **Accents:** Emerald-brand variants (10%, 20%, 80% opacity)
- **Danger:** Red (for delete actions)
